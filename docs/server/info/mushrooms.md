# Culegător de Ciuperci

## Introducere

Pentru această activitate este necesar un număr minim de **150 ore**. Pe server există câteva locații unde cresc multiple tipuri de ciuperci, acestea se pot vedea pe harta printr-un blip specific.

Pentru a le putea culege trebuie să aveți echipat un **Cuțit de ciuperci**.

În oraș veți găsi doar ciuperci de **zi**, indiferent de timpul zilei (zi sau noapte). Iar cele de **noapte** vor crește pe un nou teritoriu sindicat, atât ziua cât și noaptea. Mai multe detalii aflați IC.

În funcție de vremea de pe server acestea cresc diferit (mai rapid sau mai lent).

Aceste ciuperci pot fi folosite atât pentru anumite droguri, cât și pentru mâncare.
