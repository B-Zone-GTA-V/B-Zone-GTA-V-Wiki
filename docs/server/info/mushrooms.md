# Culegător de Ciuperci

## Introducere

Pentru această activitate este necesar un număr minim de **150 ore**. Pe server există câteva locații unde cresc multiple tipuri de ciuperci.

Acestea sunt diferite în funcție de ora din zi, sunt ciuperci de zi și ciuperci de noapte. Pentru a le putea culege trebuie să aveți echipat un **Cuțit de ciuperci**.

În funcție de vremea de pe server acestea cresc diferit (mai rapid sau mai lent).

Aceste ciuperci pot fi folosite atât pentru noul drog, cât și pentru mâncare.
